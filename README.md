## Requirements 

- PHP >= 7.4.0

## Installation

Just clone the project to anywhere in your computer. 
Run ` composer install ` <br>
and ` php artisan migrate `
php artisam db:seed 

Now you are done. 
<br>

` php artisan serve ` and open the project on the browser.
